# 🎮 Simon Says Game

A fun memory game built using HTML, CSS, and JavaScript.

## Features
- Interactive gameplay
- Random color sequence generation
- Level progression
- Score tracking
- Responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Play
1. Press any key to start.
2. Watch the color sequence.
3. Repeat the sequence by clicking the buttons.
4. The sequence gets longer every level.
5. The game ends if you click the wrong button.
